Cyberdeck display system for ESP32-S3 color LCD panels, configured entirely in Lua.

* Features

- 🖥️ *800×480 RGB565 Color Display* - Full color LCD support
- 🔧 *Lua Configuration* - Define pages and widgets in Lua
- 🧩 *Widget System* - Create custom widgets with easy-to-use APIs
- 📶 *WiFi Connectivity* - Fetch data from APIs
- 👆 *Touch & Gestures* - Swipe between pages, tap to interact
- 🦀 *Built with Rust* - Safe, fast, and reliable

* Display

- [[https://www.amazon.com/ELECROW-Dual-Core-Compatible-PlatformIO-MicroPython/dp/B0DL5WB7RQ/][Elecrow ESP32 5-inch Display]] (800×480 RGB565)

* Prerequisites

- [[https://nixos.org/download.html][Nix]] with flakes enabled
- USB connection to device

* Quick Start

** 1. Enter Development Environment

#+begin_src sh
nix develop
#+end_src

** 2. Configure (Optional)

#+begin_src sh
cp .env.example .env
# Edit .env with your WiFi credentials and API keys
#+end_src

** 4. Build

#+begin_src sh
cargo build --release -p moondeck-app
#+end_src

** 5. Flash to Device

#+begin_src sh
espflash flash target/xtensa-esp32s3-espidf/release/moondeck --partition-table target/xtensa-esp32s3-espidf/release/partition-table.bin
#+end_src

** 6. Monitor Output

#+begin_src sh
espflash monitor
#+end_src

* Live Reload

Edit Lua widget files on your computer and push them to the device over WiFi — no re-flash needed. The device IP is shown on the display during boot.

** Deploy Changed Files

Uploads only git-changed config files and triggers a reload:

#+begin_src sh
nu scripts/deploy.nu <device-ip>
#+end_src

** Deploy All Files

#+begin_src sh
nu scripts/deploy.nu <device-ip> --all
#+end_src

** Reload Only

Trigger a reload without uploading (e.g. after manual uploads):

#+begin_src sh
nu scripts/deploy.nu <device-ip> --reload
#+end_src

* Troubleshooting

** Build Errors

#+begin_src sh
# Clean and rebuild
cargo clean
nix develop
cargo build --release
#+end_src

** WiFi Not Connecting

- Check =.env= file exists and has correct credentials
- Ensure 2.4GHz network (ESP32 doesn't support 5GHz)
